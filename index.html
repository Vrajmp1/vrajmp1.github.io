Some of My Projects

Matching Game

I created a matching game using HTML, CSS, and JavaScript with a SpongeBob SquarePants theme. The game offers three modes of difficulty and tracks the number of flips as well as the best score. Click here to play the game!

Ekadashi Reminder

This simple web app, intended for mobile use is incredibly useful for Hindus, for whom fasting is required twice a month, on the 11th day ("Ekadashi") of every 15 day lunar cycle (waning and waxing). Easy to navigate and pleasing to the eye, the Ekadashi Reminder web app is simple to manuever and saves Hindus the extra effort of finding a Hindu religous calendar every time they need to know when to plan for a fast. It also includes a private birthday database, with the same format.Click here to visit this project.

Expense Tracker

Using the tkinter GUI toolkit, I built a powerful expense tracker application. The app features a simple way to enter expense name, type, date, and amount on the main screen and saves by writing to a csv file. A user can also display all expenses directly in the application and make modifications. The user has the option to navigate to the statistics screen which allows a quick and easy summary of expenses based on the filters the user chooses to place, such as type of expense and date range. Emphasis given on functionality in this project. Click here to view source code.
